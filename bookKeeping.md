navigating cli

  mac                  |    windows
  -----------------------------------
  mkdir <file name>    |    mkdir <file name>
  cd <folder name>     |    cd <file name>
  cd ../../../         |    cd ..
  ls                   |    dir
  touch <file name>    |    echo .>> file_name (or)  New-Item -ItemType file_name
  rm <file name>       |    del <file name>
  rm -rf <folder name> |    rm dir
  cd ~/Desktop         |    cd desktop
  pwd                  |    cd
